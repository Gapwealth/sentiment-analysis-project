📊 Sentiment Analysis Project

 📌 Description

This project analyzes whether text is positive, negative, or neutral.

🚀 Features

* Uses TextBlob NLP
* Simple CLI tool
* Fast and beginner-friendly

▶️ How to Run

1. Install dependency:

```
pip install textblob
```

2. Run:

```
python sentiment.py
```
